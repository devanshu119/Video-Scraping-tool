Video Scraping Tool

A lightweight Python-based tool for automatically downloading audio from
every video in a YouTube playlist. This tool simplifies mass audio
extraction without manual downloading — perfect for lectures, podcasts,
music collections, and more.

Features: - Extracts audio from all videos inside a YouTube playlist. -
Automatically saves audio files locally. - Simple and beginner-friendly
Python script. - Uses yt-dlp (more reliable than youtube-dl). - Works on
Windows, macOS, and Linux.

Project Structure: Video-Scraping-tool/ ├── scrap.py # Main script for
playlist audio downloading ├── python script.txt # Notes / early version
of script └── README.txt # This documentation file

Requirements: - Python 3.x - Install the required library: pip install
yt-dlp

Usage: 1. Clone the repository: git clone
https://github.com/devanshu119/Video-Scraping-tool cd
Video-Scraping-tool

2.  Install dependencies: pip install yt-dlp

3.  Run the scraper: python scrap.py

(Optional) With arguments: python scrap.py –playlist –output

requirements.txt content: yt-dlp

Notes: - Use responsibly (follow YouTube TOS). - Only download content
you have rights to. - Playlist downloads depend on network speed and
playlist length.
